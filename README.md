# Página Web - VibrAR Fest

Este proyecto es una landing page promocional
desarrollada en HTML y CSS.

## Descripción
Se trata de una página web informativa
sobre un evento musical, donde se presentan
los artistas invitados, fechas, ubicación
y opciones de contacto o compra de entradas.

## Tecnologías utilizadas
- HTML5
- CSS

## Funcionalidades
- Presentación del evento
- Información sobre artistas
- Sección de fechas y ubicación
- Diseño estructurado con etiquetas semánticas
